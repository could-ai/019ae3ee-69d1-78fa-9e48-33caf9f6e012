# SSC UPSC Exam Prep App

A comprehensive Flutter application for competitive exam preparation (SSC, UPSC, CG GK, Maths, Reasoning).

## Getting Started

This project is a starting point for a Flutter application.

## How to Build the APK

As this is a source code project, you need to compile it to get the APK file.

1.  **Install Flutter**: Ensure you have the Flutter SDK installed on your machine.
2.  **Open Terminal**: Navigate to this project folder in your terminal or command prompt.
3.  **Run Build Command**: Execute the following command:

    ```bash
    flutter build apk --release
    ```

4.  **Locate APK**: Once the build finishes, you can find the APK file at:
    `build/app/outputs/flutter-apk/app-release.apk`

You can then transfer this file to your Android device to install it.

## Features

- **GK & GS**: General Knowledge and Studies
- **Mathematics**: Quantitative Aptitude
- **Reasoning**: Logical and Direction questions
- **CG GK**: Chhattisgarh specific General Knowledge
- **UPSC Special**: High-level civil services questions
